Year 7

Term 1

Impact of technology
Collaborating online respectfully


This unit introduces learners to the school network, covering appropriate usage and key online safety topics such as respecting others, identifying strangers, and understanding cyberbullying. It also teaches effective use of presentation software.

Term 2

Using Media
Gaining support for a cause


Learners build IT and digital literacy skills by creating a blog post supporting a real-world cause. They develop formatting techniques and explore legal and ethical issues around using others’ work.

Term 3

Modelling data
spreadsheets


Learners progress from basic spreadsheet knowledge to confidently modelling data. They use formulas, including COUNTIF, through engaging tasks, gaining transferable skills for computing and other subjects.

Term 4

Understanding Computers


This theory-based unit covers core concepts like input/output, the IPO model, and the Fetch-Decode-Execute cycle. Learners explore binary conversions, ASCII, binary addition, and how data is stored. The unit ends with a look at the evolution of communication devices and emerging technologies

Term 5

Control System with Flowol


In this practical unit, learners use flowchart-based software (e.g. Flowol) to create systems with loops, inputs, and outputs. They develop control solutions while building understanding of key programming concepts.

Term 6

Game Development Scratch


Pupils explore Scratch by reverse-engineering games, then plan and build their own using variables, broadcasts, lists, and operators. They create playable games with scoring, lives, and randomisation, and learn to test and debug effectively.

Year 8

Term 1

Computer Crime and Cyber Security


This unit introduces key laws (Computer Misuse Act, Data Protection Act, GDPR, Copyright) and explores phishing, hacking, identity theft, and social media safety. Pupils also learn about online privacy, health and safety, and environmental issues like e-waste.

Term 2

Networks


This theory unit introduces LANs, WANs, IP addresses, and the World Wide Web. Pupils learn about client-server and peer-to-peer networks, cloud computing, encryption, and data security. The unit ends with a multiple-choice assessment and prepares learners for GCSE-level networking.

Term 3

Graphics


Pupils are introduced to bitmap and vector graphics, learning how images are represented and stored. They practise design and photo-editing skills using a graphics package, producing posters for assessment.

Term 4

Intro to Python


Pupils learn to develop simple programs using Python, focusing on correct syntax, algorithm design, and debugging. They create a portfolio of working programs for assessment, gaining foundational coding skills without delving into object-oriented concepts

Term 5

HTML & website Development


Pupils learn to build responsive websites using HTML and CSS. They create styled pages with text, images, navigation links, and web forms. The unit covers good design principles, data privacy, and ends with a portfolio of completed websites.

Term 6

Games Creation


Pupils design and code two games using a visual programming environment. They begin with a firework game using particle emitters and events, then develop a platform game with sprites, collision detection, scoring, and variables. Object-oriented concepts like objects and instances are introduced in a KS3-friendly way, with a focus on iterative development and debugging.

Year 9

Term 1

Layers of Computing


Learners explore how computing systems work—from software and operating systems to hardware and binary. The unit simplifies complex ideas and ends with discussions on AI and open-source software, linking modern topics to core concepts.

Term 2

Data Representation


GCSE Prep – Computer Systems & Security (OCR J277 1.2.3 & 1.2.4) This unit covers six topics plus an assessment, aligned with the OCR J277 spec. It introduces key concepts in computer systems and security, with flexible lesson timing based on depth and homework review.

Term 3

Python Next Steps


Building on prior experience, pupils revisit core Python skills before exploring loops, lists, and functions. They learn modular programming and develop confidence through iterative testing. This unit prepares learners for GCSE Computing and helps confirm their interest in the subject.

Term 4

Computational Thinking


Pupils explore problem-solving through unplugged activities, learning key concepts like logic gates, decomposition, and network topology. They apply computational thinking with paper-based and digital tasks, including a Scratch activity. The unit ends with an assessment and introduces modern computing ideas in an engaging, accessible way.

Term 5

Spreadsheets Modelling


Pupils create and format financial models using Excel, revisiting core skills and learning advanced features like SUM, IF, COUNTIF, conditional formatting, validation, charting, and macros. They model TV show income, merchandising, and seating plans, exploring “what if” scenarios.

Term 6

AI and Machine Learning


Pupils explore AI concepts, history, and ethics before applying theory in Scratch projects. They adapt programs for image detection, chatbots, and review analysis using lists and subroutines. Activities are accessible and support both online and paper-based learning
